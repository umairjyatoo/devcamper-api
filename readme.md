# DevCamper API

> Backend API for the Bootcamp directory application.

# usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own.

## Install dependencies

```
npm install
```

## Run App

```
# Run in Dev mode
npm run dev

# Run in Production mode
npm start
```
